# exploiTuga Website

Official website for [exploiTuga](https://github.com/exploiTuga) - Portuguese cybersecurity community.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Structure

```text
src/
├── components/   # Reusable UI components
├── layouts/      # Page layouts
├── pages/        # Route pages
├── data/         # Static data
└── styles/       # Global CSS
```
